# Diwali Sales Analysis

##  About The Project
Exploratory Data Analysis (EDA) on 
Diwali sales data to uncover buying 
patterns, top performing products, 
and customer insights.

## Key Insights
- ✅ Identified top selling categories
- ✅ Analyzed customer demographics
- ✅ Found peak sales periods
- ✅ Regional sales comparison

1. Data Cleaning
2. Exploratory Data Analysis
